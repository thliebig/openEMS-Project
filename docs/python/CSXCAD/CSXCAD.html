CSXCAD

class CSXCAD.ContinuousStructure

With this class different properties and primitives can be created and a rectengular grid assigned.

See also

CSXCAD.CSProperties
CSXCAD.CSPrimitives
CSXCAD.CSRectGrid

Examples

Create a metal box:

>>> CSX   = CSXCAD.ContinuousStructure()
>>> metal = CSX.AddMetal('metal') # create a metal property with name "metal"
>>> start = [0,0,0]
>>> stop  = [1,2,1]
>>> box   = metal.AddBox(start, stop) # Assign a box to propety "metal"
